## Description

Inner service for sending and getting inner events. For example, when there is a query to backend, service can send events of success or fail of the query. The same message is sent after log in or log out. Using this service you can subscribe to the events and perform with it further actions.

Внутренний сервис для отправки и получения внутренних событий. Например, при запросах к беку сервис может отправлять события успешности или неуспешности запроса.
