## Description

This service contains methods that are used by other services or components to perform different tasks. It is some kind of a helper, but with more logic. This service handles incoming messages from urls, has public methods for public usage.

Example:<br>
`windowResize` method is used everywhere when we need to keep track of the browser window size change.<br>
`scrollTo` - is a ready to use logic to scroll to the block, which is used in a whole application.

## Описание

Данный сервис содержит методы, которые используются другими сервисами/компонентами для выполнения задач. Представляет собой что-то вроде хелпера, но содержит больше логики. Сервис обрабатывает входящие сообщения с урлов, имеет публичные методы для общественных нужд.

Например:<br>
Метод `windowResize` используется везде, где нужно следить за изменениями окна браузера.<br>
Метод `scrollTo` - уже готовая логика для скролла к блоку, используется по всему приложению.
