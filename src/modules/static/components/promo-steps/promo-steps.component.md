# Promo Steps Component
Компонент находиться в секции баннера на главной странице и отображается под ним.
Он выводит промо шаги из WP, которые записаны в дополнительных полях в посте со слагом **"promo-steps"**.

Шаг может быть просто текстом, ссылкой или кнопкой открывающей модальное окно.
Не может быть смешанных по типу шагов.

## Настройка промо шагов
Для создания кастомного поля нужно включить два плагина **"ACF to REST API"** и **"Advanced Custom Fields"**.
Поле может иметь произвольное название, но внутреннее поле должно иметь название **"Steps"**, тип **"Group"** и содержать в себе **n** количество шагов, которые так же имеют тип **"Group"** и могут иметь произвольное имя.

![Custom fields](../assets/promo-steps/image-1.png)

По умолчанию ожидается 3 шага. Если их больше или меньше, то нужно редактировать scss т.к. шаги выводятся в 3 колонки.

В каждом шаге должны быть обязательные поля **"title"** и **"desc"** и опционально могут быть добавлены: **sref, srefParams, href, modal**.

![Prop custom fields](../assets/promo-steps/image-2.png)

Тип шага определяется кодом ниже:
```ts
public getStepType(step: IStep): string {
        if (step.sref) {
            return 'sref';
        } else if (step.href) {
            return 'href';
        } else if (step.modal) {
            return 'modal';
        } else {
            return 'default';
        }
    }
```
