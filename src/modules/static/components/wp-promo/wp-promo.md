# wp-promo
promo output component from wp

# Входные параметры
`categorySlug` - по умолчанию стоит категория 'promotion_posts', по ней будет сделан запрос на wp, и будет получен лист значений

`modalSettings` - настройки модального окна, которое показывает полное описание

`redirectButton` - настройки кнопки, которая редирект делает

`modalButton` - настройки кнопки открытия модалки

`titleComponentParams` - настройки заголовка всего компонента



# для корректной работы на wordpress должна быть создана категория, к которой будет идти запрос и мы будем получать с нее список постов для отображения
каждый такой пост должен содержать следующие customFields:

`short_description` - строка, которая будет показывать короткое описание

`background_image` - ссылка на background изображение

`bg_gradient` - использовать ли градиент(checkbox)

`bonus_size` - 'half_screen' | 'fullscreen' какой тип элемента(на всю ширину экрана или нет)

`button_link` - селектор базовых ссылок

`custom_button_link` - json строка, которая должна содержать `sref` и `srefParams`, которые затем после парсинга будут переданы в button компонент

`button_text` - текст кнопки

`decor_image` - ссылка на декор-изображение

`promo_sort_weight` - "вес" элемента, по нему будет произведена сортировка
