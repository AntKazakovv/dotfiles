## docs
### Структура
* **![](../assets/folder.svg) compodoc**
* **![](../assets/folder.svg) sassdoc**
* index.html

### Описание
Директория сборки инлайн-документации.
* **compodoc** - директория сборки инлайн документации TypeScript. Используется Compodoc.
* **sassdoc** - директория сборки инлайн документации SASS. Используется SassDoc.
* **index.html** - используется для корректного деплоя документации. В разработке не участвует.
