# 4. Modules

* **[app](../modules/AppModule.html)** - _is the core module that contains all other modules._
* **[bonuses](../modules/BonusesModule.html)** - _provides the bonus-related functionality._
* **[core](../modules/CoreModule.html)** - _is the base module containing the entities required to run the application, as well as all shared components used in all other modules._
* **[finances](../modules/FinancesModule.html)** - _вprovides the functionality for depositing and withdrawing funds._
* **[games](../modules/GamesModule.html)** - _provides the functionality for displaying, running, filtering and searching for games._
* **[menu](../modules/MenuModule.html)** - _includes all the project-specific menus, such as the main menu, profile menu, category menu, footer menu and other menus, including mobile ones._
* **[profile](../modules/ProfileModule.html)** - _displays the user profile._
* **[promo](../modules/PromoModule.html)** - _includes all promotional materials, such as banners, recent winners, jackpot sliders, etc._
* **[sportsbook](../modules/SportsbookModule.html)** - _provides the Sportsbook management functionality._
* **[static](../modules/StaticModule.html)** - _includes static texts, images and other data obtained from WordPress._
* **[store](../modules/StoreModule.html)** - _is the loyalty store._
* **[tournaments](../modules/TournamentsModule.html)** - _provides the tournament functionality on the website._
* **[user](../modules/UserModule.html)** - _displays, processes and modifies the visitor data._
* **[livechat](../modules/LivechatModule.html)** - _provides the options for integrating chats into the application._

# 4. Модули

* **[app](../modules/AppModule.html)** - _основной модуль, содержит все прочие модули._
* **[bonuses](../modules/BonusesModule.html)** - _функционал бонусов._
* **[core](../modules/CoreModule.html)** - _базовый модуль, содержащий сущности, требующиеся для запуска приложения а
  так же все общие, используемые во всех прочих модулях, компоненты._
* **[finances](../modules/FinancesModule.html)** - _всё, что связано с вводом и выводом денег._
* **[games](../modules/GamesModule.html)** - _отвечает за вывод, запуск, фильтрацию и поиск игр._
* **[menu](../modules/MenuModule.html)** - _все меню проекта: главное меню, меню профиля, меню категорий, меню футера и прочие меню, включая мобильные._
* **[profile](../modules/ProfileModule.html)** - _отвечает за отображение профиля пользователя._
* **[promo](../modules/PromoModule.html)** - _все промо-материалы: баннеры, последние победители, слайдер джекпотов и пр._
* **[sportsbook](../modules/SportsbookModule.html)** - _управление Спортсбуком._
* **[static](../modules/StaticModule.html)** - _статичные тексты, изображения и прочие данные, получаемые из WordPress._
* **[store](../modules/StoreModule.html)** - _магазин лояльности._
* **[tournaments](../modules/TournamentsModule.html)** - _отвечает за работу турнирной системы на сайте._
* **[user](../modules/UserModule.html)** - _отвечает за отображение, обработку и изменение данных посетителя._
* **[livechat](../modules/LivechatModule.html)** - _отвечает за интеграцию чатов в приложении._
