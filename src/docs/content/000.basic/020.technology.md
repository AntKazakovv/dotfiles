# Используемые технологии

* **[Angular](https://angular.io/)** - основной фреймворк.
* **[SASS](https://sass-lang.com/)** - препроцессор стилей CSS.
* **[Gulp](https://gulpjs.com/)** - таск-менеджер для задач, выполняемых перед сборкой проекта.
* **[Gulp Angular Gettext](https://www.npmjs.com/package/gulp-angular-gettext)** - система переводов (мультиязычность
  приложения), основанная на GNU Gettext.
* **[Webpack](https://webpack.js.org/)** - стандартный сборщик Angular приложений.
* **[ESLint](https://eslint.org/)** - стандартный линтер для Angular фреймворка.
* **[UI-Router](https://ui-router.github.io/ng2/)** - роутер для фреймворка Angular.
* **[Lodash-ES](https://www.npmjs.com/package/lodash-es)** - библиотека Lodash.
* **[Luxon](https://luxon.com/ru/)** - библиотека для работы с датой и временем.
* **[Bootstrp 5](https://getbootstrap.com/docs/5.0/getting-started/introduction/)** - частичное использование
  фреймворка, только некоторый функционал, например модальные окна. CSS стили - только сетка.
* **[Bowser](https://www.npmjs.com/package/bowser)** - библиотека для работы с User-agent браузера.
* **[SASS-doc](https://sass-lang.com/documentation)** - инлайн документация SASS.
* **[StyleLint](https://stylelint.io/)** - линтер CSS.
* **[Swiper](https://swiperjs.com/)** - JS слайдер.
