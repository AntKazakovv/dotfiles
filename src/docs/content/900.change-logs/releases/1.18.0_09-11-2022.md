### SCR [#413841](https://tracker.egamings.com/issues/413841) - feat: add registration currency filter by countries

`base.registration.regCurrenciesByCountries`

Принцип работы правила: Проверяется правило для текущей страны юзера, при отсутствии такого - доступные валюты не фильтруются.

Пример
для гео rus доступна валюта при регистрации - 'RUB'
для гео deu достпные валюты при регистрации - 'EUR', 'RUB'
для гео blr доступна валюта при регистрации - 'EUR'

`regCurrenciesByCountries: {
      rus: ['RUB'],
      deu: ['EUR', 'RUB'],
      blr: ['EUR'],
 },`
