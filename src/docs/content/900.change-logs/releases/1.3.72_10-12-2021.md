### SCR [#315082](https://tracker.egamings.com/issues/315082) - feat: add games sorting direction params

Adding config params for games in category sorting direction.

Params interface:

```ts
export interface IGamesSortSetting {
    direction?: IGameSortDirection;
}
```

Config example:

```ts
export const $games: IGamesConfig = {
    categories: {
        gamesSortSetting: {
            direction: {
                sortPerLanguage: 'asc',
                sortPerCategory: 'asc',
                baseSort: 'desc',
            },
        },
    },
};
```
