### SCR [#571781](https://tracker.egamings.com/issues/571781) - feat: payment systems categories for every theme

Добавлена возможность группировки платежных систем для всех тем депозита и вывода.
Для чего параметр desktopMenuType категоризации платёжных систем расширен значением 'blocks'.
Также введен новый параметр useFor, для выборочного включения категорий в Депозите или Выводе (по умолчанию: 'deposit').
> XX.finances.config.ts
```
export const $finances: IFinancesConfig = {
    paySystemCategories: {
        use: true | false,
        useFor: 'withdraw' | 'deposit' | 'both',
        desktopMenuType: 'select' | 'menu' | 'blocks',
    },
};
```

В связи с введением нового функционала возможно изменение отображения по умолчанию ненастроенного компонента, при обнаружении нежелательных изменений в отображении, произвести настройку компонента.
