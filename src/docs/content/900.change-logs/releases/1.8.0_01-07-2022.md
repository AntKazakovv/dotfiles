For activate feature you should set $base.allowedIframeReferrers parameter with array of domain.

*Example:*
```typescript
export const $base: IBaseConfig = {
    // another required options here
    allowedIframeReferrers: [
        'metrika.yandex.ru',
    ],
};
```
