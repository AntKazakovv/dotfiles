### SCR [#471225](https://tracker.egamings.com/issues/471225) - fix: category params

для включения отображения меню категорий на мобилке необходимо в настройках layout.config.ts удалить парамерт display для
сомпонента category

```
{
        name: 'menu.wlc-category-menu',
        **display: {
            after: 768,
        },**
        params: {
            type: 'categories-menu',
            common: {
                useSliderNavigation: true,
            },
            menuParams: {
                sliderParams: {
                    swiper: {
                        spaceBetween: 5,
                    },
                },
            },
        },
    }
```
