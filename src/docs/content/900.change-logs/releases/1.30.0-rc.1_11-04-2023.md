### SCR [#462144](https://tracker.egamings.com/issues/462144) - fix: added option to disable More button in tournament-prizeboard

By default tournament prizeboard shows first 10 places and button "Show more" if needed.
If you want to show all places in tournament prizeboard, use this config in 04.modules.config.ts

```
export const $modules = {
    tournaments: {
        components: {
            'wlc-tournament-detail': {
                prizesParams: {
                    showMore: {
                        use: false, // set true to trim prizeboard
                        rowLimit: 10, // number of displayed rows
                    },
                },
            },
        },
    },
}; 
```
