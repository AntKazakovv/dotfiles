## 1.3.28 (19-07-2021)

### SCR [#240307](https://tracker.egamings.com/issues/240307) - [fix] angular up to 12, buld optimization
before:

start build: ~95 sec,
rebuild core module: ~ 22 sec

after:

start build: ~55 sec,
rebuild core module: ~ 2 - 10 sec

1. Angular обновлен до 12.1.1

2. Для dev-режима отключены все оптимизации в целях повышения скорости сборки.

3. Разделена логика LayoutService, код отвечающий за предоставление модулей, компонентов и сервисов, перенесен в InjectionServise.
Если необходимо получить сервис из другого модуля, не следует импортировать модуль напрямую использую метод InjectionServise.importModules.

Для этого следует использовать метод InjectionServise.getService(name: string). Метод принимает строку в формате "module-name.service-name". Предварительно в файле модуля
Данный способов не ломает ленивую загрузку модулей, а InjectionServise.importModules() - ломает, т.е. сторонний модуль попадает в главный чанк. По этой причине пересборка происходила так долго.

4. css-mqpacker заменен на css-mqpacker-webpack-plugin, из-за этого в консоль падает DeprecationWarning. Но пока только так, сам css-mqpacker старым способом больше подключить нельзя.

5. Angular CLI 12 использует версию sass 1.35.1, в которой нельзя отключить DeprecationWarning для внутреннего кода (можно только для кода зависимостей).
