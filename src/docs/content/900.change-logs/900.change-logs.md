### SCR [#524132](https://tracker.egamings.com/issues/524132) - feat: custom validators for deposit/withdraw amount field

How to add custom validators for deposit/withdraw form

1. Define your custom validation function in $base.forms.customValidators 

```typescript
customValidators: {
    customValidationFn: (control) => {
        return (Number(control.value) % 10) ? {'custom-error': true} : null;
    },
}
```

2. Add name of your custom validation function in $finances.fieldsSettings.amount 

```typescript
customValidators: {
    deposit: ['customValidationFn'],
    // withdraw: ['multipleWithdraw'], if needed
}        
```

3. Add translates for new error from you custom validation function to your project
