### SCR [#315082](https://tracker.egamings.com/issues/315082) - feat: add games sorting direction params

Adding config params for games in category sorting direction.

Params interface:

```ts
export interface IGamesSortSetting {
    direction?: {
        /** direction of sorting by sortPerLanguage, 'asc' by default */
        sortPerLanguage?: TSortDirection,
        /** direction of sorting by sortPerCategory, 'asc' by default */
        sortPerCategory?: TSortDirection;
        /** direction of sorting by general sort value, 'desc' by default */
        baseSort?: TSortDirection;
    }
}
```

Config example:

```ts
export const $games: IGamesConfig = {
    categories: {
        gamesSortSetting: {
            direction: {
                sortPerLanguage: 'asc',
                sortPerCategory: 'asc',
                baseSort: 'desc',
            },
        },
    },
};
```
