# 5.7. Wrapper component (WIP)

If you need to combine multiple components with one wrapper in *layout config, then use the **wlc-wrapper component**, which inherits from **layout component**, and specify a list of components you want to combine in params.

## Example:
```typescript
{
        name: 'core.wlc-wrapper',
        params: {
            class: 'footer-bottom',
            components: [
                {
                    name: 'core.wlc-logo',
                    display: {
                        after: 899,
                    },
                },
                {
                    name: 'core.wlc-disclaimer',
                },
                {
                    name: 'core.wlc-copyright',
                },
                {
                    name: 'core.wlc-license',
                    params: {
                        apgSeal,
                    },
                },
            ],
        },
    },
```
