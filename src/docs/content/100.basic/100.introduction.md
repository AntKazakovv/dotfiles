# 1. Структура документации

## Description

Раздел общей информации о работе WLC Engine. Содержит простые текстовые разделы с описанием всех необходимых для работы
понятий.

* **[1. Introduction](1.-introduction.html)** - Структура документации.
    * [1.1. Base](1.-introduction/1.1.-base.html) - Общее описание системы и используемые технологии, принципы и
      подходы.
    * [1.2. Global components](1.-introduction/1.2.-global-components.html) - Глобальная структура WLC проекта, схема
      взаимодействие Front-end и Back-end.
    * [1.3. Technology](1.-introduction/1.3.-technology.html) - Используемые технологии.
    * [1.4. Implementation](1.-introduction/1.4.-implementation.html) - Особенности реализации WLC Engine.
    * [1.5. Build](1.-introduction/1.5.-build.html) - Сборка проекта, Gulp + Webpack.
* **[2. Global structure](2.-global-structure.html)** - Структура WLC Engine.
* **[3. Config](3.-config.html)** - Документация на глобальную конфигурацию проекта WLC.
    * [3.1. Layouts](3.-config/3.1.-layouts.html) - Конфигурация базовой структуры шаблона страниц.
* **[4. Modules](4.-modules.html)** - Описание модульной системы, особенности настройки и загрузки модулей.
    * [4.1. Structure](4.-modules/4.1.-structure.html) - Модульная структура.
* **[5. Components](5.-components.html)** - Описание структуры компонентов, принципа их построения и конфигурации.
