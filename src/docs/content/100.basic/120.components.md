# 1.2. System components

## Front-end

* **WLC-Engine** - is the engine itself. It's an NPM package which includes all the files required for the proper operation of your project. It is installed as an NPM package in your project using packages.json.
* **WLC** - is the program code of a given project. It contains the configuration files of your project, such as images, styles and HTML templates for the project customization. During the assembly process, it uses both local files and engine files (as local ones using symbolic links).
* **Messages** - contains the localization files. The project is hosted in a dedicated repository and can be included as a git submodule.

## Back-end

* **WLC-core** - is the core PHP engine that communicates with the core back-end system (fundist).
* **Fundist** - is the core back-end system. It is responsible for the processing of all user data.
* **Loyalty** - is the back-end system which is responsible for the processing of loyalty data (bonuses, competitions, etc.). It can be managed via fundist.
* **Affiliate** - is the back-end system which is responsible for the processing of affiliate data. It can be managed via fundist.

# 1.2. Компоненты системы

## Front-end

* **WLC-Engine** - сам движок. Представляет собой NPM пакет, включающий в себя все необходимые файлы для работы проекта.
  Устанавливается в проекте как NPM пакет через packages.json.
* **WLC** - код конкретного проекта. Содержит файлы конфигурации проекта, а так же файлы изображений, стилей и HTML
  шаблонов, необходимые для индивидуальной настройки проекта. При сборке использует как локальные файлы, так и файлы
  движка как локальные (по символическим ссылкам).
* **Messages** - файлы локализации. Проект расположен в отдельном репозитории, подключается как git submodule.

## Back-end

* **WLC-core** - основной PHP движок, осуществляющий общение с бекенд-системой fundist.
* **Fundist** - основная бекенд-система. Осуществляет всю обработку данных пользователей.
* **Loyalty** - бекенд-система, осуществляющая обработку данных лояльности (бонусы, турниры). Управляется через fundist.
* **Affiliate** - бекенд-система, осуществляющая обработку данных аффилейтов. Управляется через fundist.
