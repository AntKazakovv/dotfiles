# 8.2. Adding fonts to your website

You can add custom fonts using the `initFont` mix-in. This mix-in has the two parameters:

* The first parameter is the configuration to be used for generating the font prefix. Example of the configuration:  

**This configuration is intended for generator: https://google-webfonts-helper.herokuapp.com**
```typescript
$fontConfig: (
    weight: (
        100: '100',
        200: '200',
        300: '300',
        400: 'regular',
        500: '500',
        600: '600',
        700: '700',
        900: '900',
    ),
    style: (
        normal: '',
        italic: 'italic',
    ),
    custom: (
        '400/italic': 'italic',
    )
);
```
Prefixes are generated in the following format: (alias weight)-(alias style). You can also generate a custom prefix for a given font by specifying weight/style as a key in the `custom` field and passing the required value.

* The second parameter in the configuration for a given font, e.g. the example below illustrates the configuration for the Inter font:
```typescript
$inter: (
    name: 'Inter',
    weight: (100, 300, 400, 500, 600),
    style: (normal),
    format: ('woff2', 'woff'),
    path: '/gstatic/wlc/fonts/Inter/inter-v3-vietnamese_latin-ext_latin_greek-ext_greek_cyrillic-ext_cyrillic',
);
```
Please note that the formats should be listed starting from the least supported font. In addition, the font name must contain the path: **path-to-font/font-name**.   
The mix-in also supports adding dynamic fonts. To use this feature, add the `variations` format and your font with the `-variable.woff2` prefix.

## Use case:
**src/engine-scss/base/_fonts.scss**
```typescript
$fontConfig: (
    weight: (
        100: '100',
        200: '200',
        300: '300',
        400: 'regular',
        500: '500',
        600: '600',
        700: '700',
        900: '900',
    ),
    style: (
        normal: '',
        italic: 'italic',
    ),
    custom: (
        '400/italic': 'italic',
    )
);
$roboto: (
    name: 'Roboto',
    weight: (100, 300, 400, 500, 700, 900),
    style: (normal, italic),
    format: ('woff2', 'woff'),
    path: '/gstatic/wlc/fonts/Roboto/roboto-v20-vietnamese_latin-ext_latin_greek-ext_greek_cyrillic-ext_cyrillic',
);
 
@include initFont($fontConfig, $roboto);
```
We support 5% of the newest browser versions, which means that you can only include the woff2 format (https://caniuse.com/?search=woff2).

# 8.2. Добавление шрифтов на сайт

Шрифты добавляются с помощью миксина `initFont`. Миксин имеет два параметра:

* Первым параметром передается конфиг для генерации префикса шрифта. Пример конфига:   

**Данный конфиг написан для генератора: https://google-webfonts-helper.herokuapp.com**
```typescript
$fontConfig: (
    weight: (
        100: '100',
        200: '200',
        300: '300',
        400: 'regular',
        500: '500',
        600: '600',
        700: '700',
        900: '900',
    ),
    style: (
        normal: '',
        italic: 'italic',
    ),
    custom: (
        '400/italic': 'italic',
    )
);
```
Префикс генерируется по маске **(алиас weight)-(алиас style)**. Также есть возможность сгенерировать кастомный префикс для конкретного шрифта, в поле `custom` указав в качестве ключа **weight/style** и передав необходимое значение.

* Вторым параметром передается конфиг для конкретного шрифта, например ниже представлен конфиг для шрифта Inter:
```typescript
$inter: (
    name: 'Inter',
    weight: (100, 300, 400, 500, 600),
    style: (normal),
    format: ('woff2', 'woff'),
    path: '/gstatic/wlc/fonts/Inter/inter-v3-vietnamese_latin-ext_latin_greek-ext_greek_cyrillic-ext_cyrillic',
);
```
Нужно учитывать, что перечисление форматов начинается с шрифта с наименьшей поддержкой. И путь указывается вместе с названием шрифта: **path-to-font/font-name**.  
Миксин также поддерживает добавление вариативных шрифтов. Для этого в конфиге шрифта нужно добавить формат `variations` и шрифт с префиксом `-variable.woff2`.

## Пример использования:
**[src/engine-scss/base/_fonts.scss]()**
```typescript
$fontConfig: (
    weight: (
        100: '100',
        200: '200',
        300: '300',
        400: 'regular',
        500: '500',
        600: '600',
        700: '700',
        900: '900',
    ),
    style: (
        normal: '',
        italic: 'italic',
    ),
    custom: (
        '400/italic': 'italic',
    )
);
$roboto: (
    name: 'Roboto',
    weight: (100, 300, 400, 500, 700, 900),
    style: (normal, italic),
    format: ('woff2', 'woff'),
    path: '/gstatic/wlc/fonts/Roboto/roboto-v20-vietnamese_latin-ext_latin_greek-ext_greek_cyrillic-ext_cyrillic',
);
 
@include initFont($fontConfig, $roboto);
```
Мы поддерживает 5% последних браузеров, а значит можно подключать только формат **woff2** (https://caniuse.com/?search=woff2).
