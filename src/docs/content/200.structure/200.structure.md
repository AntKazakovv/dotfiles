# 2. Файловая структура WLC-Engine

## wlc-engine

* **![](../assets/folder.svg) docs**
    * **![](../assets/folder.svg) compodoc** - _директория сборки инлайн документации TypeScript.
      Используется [Compodoc](https://compodoc.app/)._
    * **![](../assets/folder.svg) sassdoc** - _директория сборки инлайн документации SASS.
      Используется [SassDoc](http://sassdoc.com/)._
    * index.html - _используется для корректного деплоя документации. В разработке не участвует._
* **![](../assets/folder.svg) e2e** - _директория тестов End-to-end._
* **![](../assets/folder.svg) gulpfile**
    * **![](../assets/folder.svg) global-config** - _Шаблоны JSON файлов и другие файлы конфигурации._
    * **![](../assets/folder.svg) tasks** - _файлы gulp задач. Задачи сгруппированы по назначению в соответствующие
      файлы._
    * config.js - _файл конфигурации сборщика Gulp._
    * gulpfile.js - _основной файл Gulp._
* **![](../assets/folder.svg) src**
    * **![](../assets/folder.svg) assets** - _директория файлов, копируемых в проект при сборке. В настоящий момент не
      используется._
    * **![](../assets/folder.svg) docs** - _исходники данной документации._
    * **![](../assets/folder.svg) engine-scss** - _исходники стилей WLC Engine._
* **![](../assets/folder.svg) utils** - _Дополнительные скрипты, использующиеся при деплое WLC Engine и других
  операциях._
* **![](../assets/folder.svg) wlc-src** - _файлы, используемые при локальной сборке WLC Engine для юнит-тестов.
  Представляют собой заглушки-дубликаты файлов проекта._
* .editorconfig - _настройки [EditorConfig](https://editorconfig.org/)._
* .eslintrc.js - _конфигурация [EsLint](https://eslint.org/)._
* .gitignore - _список исключений для [Git](https://git-scm.com/docs/gitignore)._
* .gitmodules - _данные о [git-submodule](https://git-scm.com/docs/git-submodule)._
* .huskyrc.js - _настройки пакета [Husky](https://typicode.github.io/husky/#/)._
* angular.json - _настройки сборки приложения [Angular](https://angular.io/guide/workspace-config)._
* browserslist - _список используемых браузеров для выбора режима сборки_
* gulpfile.js - _главный файл
  таск-менеджера [Gulp](https://gulpjs.com/docs/en/getting-started/javascript-and-gulpfiles/)._
* karma.conf.js - _настройка unit-тестов [Karma](https://karma-runner.github.io/latest/index.html)._
* make_release.sh - _скрипт автоматизации деплоя релизной версии WLC Engine. Для выполнения необходимо иметь
  соответствующий уровень доступа._
* package.json - _зависимости [NPM: package.json](https://docs.npmjs.com/cli/v7/configuring-npm/package-json)._
* package-lock.json - _лок-файл [NPM: package-lock.json](https://docs.npmjs.com/cli/v7/configuring-npm/package-lock-json)._
* README.md - _Описание проекта WLC Engine._
* stylelint.config.js - _конфигурация линтера [StyleLint](https://stylelint.io/user-guide/configure)._
* tsconfig.app.json - _настройка сборщика TypeScript для приложения WLC._
* tsconfig.docs.json - _настройка сборщика TypeScript для документации._
* tsconfig.inline.json - _настройка сборщика TypeScript для инлайн-файлов._
* tsconfig.json - _общая настройка сборщика TypeScript. Наследуется остальными файлами конфигурации._
* tsconfig.spec.json - _настройка сборщика TypeScript для юнит-тестов._
* tsconfig.worker.json - _настройка сборщика TypeScript для воркеров._
