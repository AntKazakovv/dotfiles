# 2. Файловая структура WLC-Engine

## wlc-engine

* **![](../assets/folder.svg) docs**
    * **![](../assets/folder.svg) compodoc** - _directory for compiling the inline TypeScript documentation. The documentation is compiled using [Compodoc](https://compodoc.app/)._
    * **![](../assets/folder.svg) sassdoc** - _the directory for compiling the inline SASS documentation. The documentation is compiled using [SassDoc](http://sassdoc.com/)._
    * index.html - _used for the proper deployment of the documentation. It is not used in the development process._
* **![](../assets/folder.svg) e2e** - _the directory for End-to-end tests_
* **![](../assets/folder.svg) gulpfile**
    * **![](../assets/folder.svg) global-config** - _templates of JSON files, as well as other configuration files._
    * **![](../assets/folder.svg) tasks** - _files of gulp tasks. The tasks are grouped by their designation into the corresponding files._
    * config.js - _the configuration file of the Gulp builder._
    * gulpfile.js - _the main Gulp file_
* **![](../assets/folder.svg) src**
    * **![](../assets/folder.svg) assets** - _the directory for files to be copied to the project during the assembly process. It is not used in the current version._
    * **![](../assets/folder.svg) docs** - _the source files of the documentation._
    * **![](../assets/folder.svg) [engine-scss](7.-styles.html)** - _the source files of the WLC Engine styles._
    * **![](../assets/folder.svg) modules** - _the source code of the WLC Engine modules._
    * **![](../assets/folder.svg) svg** - _the source code of the SVG images._
    * **![](../assets/folder.svg) system** - _additional files used by WLC Engine._
* **![](../assets/folder.svg) utils** - _additional scripts to be used during the WLC Engine deployment and for other operations._
* **![](../assets/folder.svg) wlc-src** - _files to be used during the local assembly of WLC Engine for unit tests. These are stubs that duplicate the project files._
* .editorconfig - _settings [EditorConfig](https://editorconfig.org/)._
* .eslintrc.js - _configuration [EsLint](https://eslint.org/)._
* .gitignore - _a list of exceptions for [Git](https://git-scm.com/docs/gitignore)._
* .gitmodules - _the [git-submodule](https://git-scm.com/docs/git-submodule).data_
* .huskyrc.js - _the [Husky](https://typicode.github.io/husky/#/) package settings._
* angular.json - _the assembly settings for the [Angular](https://angular.io/guide/workspace-config)._
* browserslist - _a list of supported browsers for different assembly modes_
* gulpfile.js - _the main file of the [Gulp](https://gulpjs.com/docs/en/getting-started/javascript-and-gulpfiles/)._
* karma.conf.js - _the unit test settings for [Karma](https://karma-runner.github.io/latest/index.html)._
* make_release.sh - _the automatic deployment script for the release version of WLC Engine. You will need the corresponding access rights to launch it._
* package.json - _the NPM dependencies: [package.json](https://docs.npmjs.com/cli/v7/configuring-npm/package-json)._
* package-lock.json - _
  lock file [NPM: package-lock.json](https://docs.npmjs.com/cli/v7/configuring-npm/package-lock-json)._
* README.md - _the description of the WLC Engine project._
* stylelint.config.js - _the [StyleLint](https://stylelint.io/user-guide/configure) linter configuration_
* tsconfig.app.json - _the TypeScript builder settings for the WLC application._
* tsconfig.docs.json - _the TypeScript builder settings for the documentation._
* tsconfig.inline.json - _the TypeScript builder settings for the inline files._
* tsconfig.json - _the general settings of the TypeScript builder. It is inherited by other configuration files._
* tsconfig.spec.json - _the TypeScript builder settings for the unit tests._
* tsconfig.worker.json - _the TypeScript builder settings for the workers._

# 2. Файловая структура WLC-Engine

## wlc-engine

* **![](../assets/folder.svg) docs**
    * **![](../assets/folder.svg) compodoc** - _директория сборки инлайн документации TypeScript.
      Используется [Compodoc](https://compodoc.app/)._
    * **![](../assets/folder.svg) sassdoc** - _директория сборки инлайн документации SASS.
      Используется [SassDoc](http://sassdoc.com/)._
    * index.html - _используется для корректного деплоя документации. В разработке не участвует._
* **![](../assets/folder.svg) e2e** - _директория тестов End-to-end._
* **![](../assets/folder.svg) gulpfile**
    * **![](../assets/folder.svg) global-config** - _Шаблоны JSON файлов и другие файлы конфигурации._
    * **![](../assets/folder.svg) tasks** - _файлы gulp задач. Задачи сгруппированы по назначению в соответствующие
      файлы._
    * config.js - _файл конфигурации сборщика Gulp._
    * gulpfile.js - _основной файл Gulp._
* **![](../assets/folder.svg) src**
    * **![](../assets/folder.svg) assets** - _директория файлов, копируемых в проект при сборке. В настоящий момент не
      используется._
    * **![](../assets/folder.svg) docs** - _исходники данной документации._
    * **![](../assets/folder.svg) [engine-scss](7.-styles.html)** - _исходники стилей
      WLC Engine._
    * **![](../assets/folder.svg) modules** - _исходный код модулей WLC Engine._
    * **![](../assets/folder.svg) svg** - _исходники код SVG изображений._
    * **![](../assets/folder.svg) system** - _дополнительные файлы WLC Engine._
* **![](../assets/folder.svg) utils** - _Дополнительные скрипты, использующиеся при деплое WLC Engine и других
  операциях._
* **![](../assets/folder.svg) wlc-src** - _файлы, используемые при локальной сборке WLC Engine для юнит-тестов.
  Представляют собой заглушки-дубликаты файлов проекта._
* .editorconfig - _настройки [EditorConfig](https://editorconfig.org/)._
* .eslintrc.js - _конфигурация [EsLint](https://eslint.org/)._
* .gitignore - _список исключений для [Git](https://git-scm.com/docs/gitignore)._
* .gitmodules - _данные о [git-submodule](https://git-scm.com/docs/git-submodule)._
* .huskyrc.js - _настройки пакета [Husky](https://typicode.github.io/husky/#/)._
* angular.json - _настройки сборки приложения [Angular](https://angular.io/guide/workspace-config)._
* browserslist - _список используемых браузеров для выбора режима сборки_
* gulpfile.js - _главный файл
  таск-менеджера [Gulp](https://gulpjs.com/docs/en/getting-started/javascript-and-gulpfiles/)._
* karma.conf.js - _настройка unit-тестов [Karma](https://karma-runner.github.io/latest/index.html)._
* make_release.sh - _скрипт автоматизации деплоя релизной версии WLC Engine. Для выполнения необходимо иметь
  соответствующий уровень доступа._
* package.json - _зависимости [NPM: package.json](https://docs.npmjs.com/cli/v7/configuring-npm/package-json)._
* package-lock.json - _
  лок-файл [NPM: package-lock.json](https://docs.npmjs.com/cli/v7/configuring-npm/package-lock-json)._
* README.md - _Описание проекта WLC Engine._
* stylelint.config.js - _конфигурация линтера [StyleLint](https://stylelint.io/user-guide/configure)._
* tsconfig.app.json - _настройка сборщика TypeScript для приложения WLC._
* tsconfig.docs.json - _настройка сборщика TypeScript для документации._
* tsconfig.inline.json - _настройка сборщика TypeScript для инлайн-файлов._
* tsconfig.json - _общая настройка сборщика TypeScript. Наследуется остальными файлами конфигурации._
* tsconfig.spec.json - _настройка сборщика TypeScript для юнит-тестов._
* tsconfig.worker.json - _настройка сборщика TypeScript для воркеров._
