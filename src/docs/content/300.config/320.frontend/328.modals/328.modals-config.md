# 3.2.8 Modal settings

Modal settings

**Restrict modal `'restrictModal'`**

Show error message by key and value in config

`'signup'` - Modal key

`baseConfigKey` - Config key

`baseConfigValue` - Config value

`message` - Message

`wlcElement` - wlcElement

```ts
$modals = {
    'restrictModal': <IRestrictModal>{
        'signup': {
            baseConfigKey: '$base.site.restrictRegistration',
            baseConfigValue: true,
            message: gettext('Sorry, registration is disabled.'),
            wlcElement: 'registration-is-disabled',
        },
    }
}
```

---

**Custom modal `'customModals'`**

Create a custom modal on the project

```ts
$modals = {
    'customModals': <IModalList>{
        'ctm-search': {
            config: {
                id: 'ctm-search',
                modifier: 'ctm-search',
                componentName: 'games.wlc-search',
                showFooter: false,
            },
        },
    }
}
```

---

**Opening a modal window when switching to a state `'IStatesModal'`**

`'app.home` - State name

`modal` - Modal name

`auth` - Show / not show to logged in or show to everyone `'any'`  Type: boolean | 'any'

`once` - Show on first visit or permanently

```ts
$modals = {
    'states': <IStatesModal>{
        'app.home': {
            'modal': 'login',
            'auth': false,
            'once': true,
        },
    }
}
```

# 3.2.8 Настройки модальных окон

Настройки модальных окон

**Запрет окон `'restrictModal'`**

По ключу и значению в конфиге показать сообщение об ошибке

`'signup'` - Ключ модального окна

`baseConfigKey` - Ключ в конфиге

`baseConfigValue` - Значение в конфиге

`message` - Сообщение

`wlcElement` - wlcElement

```ts
$modals = {
    'restrictModal': <IRestrictModal>{
        'signup': {
            baseConfigKey: '$base.site.restrictRegistration',
            baseConfigValue: true,
            message: gettext('Sorry, registration is disabled.'),
            wlcElement: 'registration-is-disabled',
        },
    }
}
```

---

**Кастомные мадальные окна `'customModals'`**

Создать кастомную модалку на проекте

```ts
$modals = {
    'customModals': <IModalList>{
        'ctm-search': {
            config: {
                id: 'ctm-search',
                modifier: 'ctm-search',
                componentName: 'games.wlc-search',
                showFooter: false,
            },
        },
    }
}
```

---

**Открытие модального окна при переходе на стейт `'IStatesModal'`**

`'app.home` - Имя стейта

`modal` - Имя модального окна

`auth` - Показывать / не показывать авторизованым или показать всем `'any'`  Тип: boolean | 'any'

`once` - Показывать при первом заходе или постоянно

```ts
$modals = {
    'states': <IStatesModal>{
        'app.home': {
            'modal': 'login',
            'auth': false,
            'once': true,
        },
    }
}
```
