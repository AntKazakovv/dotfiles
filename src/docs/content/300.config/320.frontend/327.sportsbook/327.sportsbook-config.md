# 3.2.7 Betradar sportsbook settings

Use the following settings to use Sportsbook

configuration example in [/config/frontend/06.sportsbook.config.ts]()

`cssfile: static/config/fileName.css` - gets the path to the style file in sportsbook

`configFile: static/config/fileName.js` - gets the path to the js file with sportsbook configurations

`theme: v2` - the theme of the sports book that will be used by default. If there is no field, the first topic will be selected.To use the second theme, you need to register the value "v2".

``` 
export const $sportsbook: ISportsbookConfig = {
    betradar: {
        cssFile: '/static/config/trebolplay.css',
        configFile: '/static/config/trebolplay.js',
        theme: 'v2',
    },
};
```

# 3.2.7 Настройки спортсбука Betradar

Настройки для использования спортбука:

Конфигурационный файл находится в проекте в [/config/frontend/06.sportsbook.config.ts]()

`cssfile: static/config/fileName.css` - указываем путь до css файла нужного спортбука

`configFile: static/config/fileName.js` - указываем путь до конфигурационного файлa для нужного спортбука

`theme: v2` - Выбираем нужную тему для спортбука. Если нам нужна 1 тема спортбука, это поле можно опустить, первая тема - дефолтная.Если нужно изпользовать вторую тему, то нужно в конфиг прописать theme: 'v2'.

``` 
export const $sportsbook: ISportsbookConfig = {
    betradar: {
        cssFile: '/static/config/trebolplay.css',
        configFile: '/static/config/trebolplay.js',
        theme: 'v2',
    },
};
```
