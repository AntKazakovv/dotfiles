# 3.2.6. Managing menus and categories using fundist

## Introduction

To set up project-wide menus and categories flexibly, fundist provides a dedicated interface, which allows you to edit menus and categories with zero coding.

The two options are available:

1. Fundist-based management
2. Simulation of the fundist-based management using the project configuration

## How fundist retrieves settings

Menu settings and category settings are retrieved using the bootstrap request with the two keys:

1. `menuSettings` (information for setting up menus)
2. `categories` (information for setting up categories)

The project will switch to maintain the new category architecture if the `categories` key has been received. Similarly, all menus switch to the new settings when the `menuSettings` key has been indicated.

# 3.2.6. Управление меню и категориями через fundist

## Введение

Чтобы не жестко на уровне проекта задавать настройки меню и категорий, был разработан интерфейс в fundist,
который позволит редактировать меню и категории без участия разработчиков.

Доступны два варианта:

1. Управление через fundist
2. Имитация управления через fundist с использованием конфигурации проекта

## Как fundist присылает настройки

Настройки меню и категорий приходят через запрос bootstrap по двум ключам

1. `menuSettings` (информация для настройки меню)
2. `categories` (информация для настройки категорий)

Далее проект переключается на поддержку новой архитектуры категорий, если получен ключ `categories`.
Аналогично автоматически все меню переключаются на новый вариант настройки, если есть ключ `menuSettings`.
