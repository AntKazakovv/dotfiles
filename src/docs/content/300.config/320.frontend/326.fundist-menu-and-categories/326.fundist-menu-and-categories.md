# 3.2.6. Управление меню и категориями через fundist

## Введение

Чтобы не жестко на уровне проекта задавать настройки меню и категорий, был разработан интерфейс в fundist,
который позволит редактировать меню и категории без участия разработчиков.

Доступны два варианта:

1. Управление через fundist
2. Имитация управления через fundist с использованием конфигурации проекта

## Как fundist присылает настройки

Настройки меню и категорий приходят через запрос bootstrap по двум ключам

1. `menuSettings` (информация для настройки меню)
2. `categories` (информация для настройки категорий)

Далее проект переключается на поддержку новой архитектуры категорий, если получен ключ `categories`.
Аналогично автоматически все меню переключаются на новый вариант настройки, если есть ключ `menuSettings`.
