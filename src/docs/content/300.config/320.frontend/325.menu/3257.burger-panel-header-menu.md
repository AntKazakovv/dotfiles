#  3.2.5.7. Menu in the header of the left burger menu 

By default, it is a set of 3 buttons ("Favorites", "Last played" and "Home") and automatically
displayed if all menus are managed through fundist (in this case, upon request `/api/v1/boostrap`
the response has a `menuSettings` key containing menu settings)

Parameter interface [IBurgerPanelHeaderMenu](../../../../interfaces/IBurgerPanelHeaderMenu.html) 

## Options

- **use** - Shows/hides the menu

- **enableByFundistMenuSettings** - if set to true, the menu will be displayed if available 
the menuSettings key in response to the `/api/v1/boostrap` request (through the menuSettings key, the fundist transfers 
settings of all project menus)

- **menuParams** - default settings for the `wlc-menu` component, based on which this menu will be displayed

- **icons.folder** - the folder on gstatic or in the project, in which the icons for this menu are located. Default 
the folder on gstatic `wlc/icons/categories/v2` is used

- **items** - menu item settings. By default, buttons without names, since they are just buttons with icons
  
```ts
burgerPanel: {
        left: {
            headerMenu: {
                use: true,
                enableByFundistMenuSettings: true,
                menuParams: {
                    type: 'burger-panel-header',
                    items: [],
                    common: {
                        useSwiper: false,
                    },
                },
                icons: {
                    folder: 'wlc/icons/categories/v2',
                },
                items: [
                    {
                        name: '',
                        type: 'sref',
                        icon: 'lobby',
                        class: 'lobby',
                        params: {
                            state: {
                                name: 'app.home',
                            },
                        },
                    },
                    {
                        name: '',
                        type: 'sref',
                        icon: 'favourites',
                        class: 'favourites',
                        params: {
                            state: {
                                name: 'app.catalog',
                                params: {
                                    category: 'favourites',
                                },
                            },
                        },
                        auth: true,
                    },
                    {
                        name: '',
                        type: 'sref',
                        icon: 'last-played',
                        class: 'last-played',
                        params: {
                            state: {
                                name: 'app.catalog',
                                params: {
                                    category: 'lastplayed',
                                },
                            },
                        },
                        auth: true,
                    },
                ],
            },
        },
    }
```

#  3.2.5.7. Меню в шапке левого бургер меню 

По дефолту представляет из себя набор из 3 кнопок ("Избранное", "Последние сыгранные" и "Домой") и автоматически
отображается в том случае, если управление всеми меню происходит через fundist (в этом случае по запросу `/api/v1/boostrap`
в ответе есть ключ `menuSettings`, содержащий настройки меню)

Интерфейс параметров [IBurgerPanelHeaderMenu](../../../../interfaces/IBurgerPanelHeaderMenu.html) 

## Параметры

- **use** - Показывает/скрывает меню

- **enableByFundistMenuSettings** - если выставлено true, меню отобразится при условии наличия
ключа `menuSettings` в ответе на запрос `/api/v1/boostrap` (через ключ `menuSettings` fundist передает
настройки всех меню проекта)

- **menuParams** - дефолтные настройки для компонента `wlc-menu`, на основе которого будет отображено данное меню

- **icons.folder** - папка на gstatic или в проекте, в которой расположены иконки для данного меню. По умолчанию
используется папка на gstatic `wlc/icons/categories/v2`

- **items** - настройки элементов меню. По дефолту кнопки без названий, так как представляют из себя просто кнопки с иконками
