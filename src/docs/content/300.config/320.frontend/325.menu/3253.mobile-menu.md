# 3.2.5.3. Mobile menu

## List of available menu items
The [src/modules/menu/system/config/mobile-menu.items.config.ts]() file contains the `wlcMobileMenuItemsGlobal` constant with button aliases. Each alias uses own settings for menu buttons.

```typescript
export const wlcMobileMenuItemsGlobal: MenuParams.IMenuItemsGlobal = {
    'mobile-menu:sportsbook': {
        name: gettext('Sportsbook'),
        type: 'sref',
        icon: 'sportsbook',
        class: 'sportsbook',
        params: {
            state: {
                name: 'app.sportsbook',
                params: {},
            },
        },
    },
    'mobile-menu:promotions': {
        name: gettext('Promotions'),
        type: 'sref',
        icon: 'promotions',
        class: 'promotions',
        params: {
            state: {
                name: 'app.promotions',
                params: {},
            },
        },
    },
```

## The default menu configuration
You can use the button aliases to build the configuration of the main menu. The [src/modules/menu/system/config/menu.config.ts]() file stores the default configuration of the mobile menu.
```typescript
import {IMenuConfig} from '../interfaces/menu.interface';
export const menuConfig: IMenuConfig = {    
    mobileMenu: {
        items: [
            'mobile-menu:sportsbook',
            'mobile-menu:promotions',
            {
                parent: 'mobile-menu:info',
                items: [
                    'mobile-menu:privacy-policy',
                    'mobile-menu:responsible-game',
                    'mobile-menu:fair-play',
                    'mobile-menu:games-rules',
                    'mobile-menu:terms-and-conditions',
                ],
            },
        ],
        icons: {
            folder: 'wlc/icons/european/v1',
            use: false,
        },
    },
};
```

## Reassigning the mobile menu in your project
**Important:**
 You don't need to set up the display of the categories (e.g. **Casino** or **Live Casino**) in the mobile menu. These buttons will be created automatically based on the category information from **fundist** and inserted in the top part of the menu.

The menu can be reassigned in your project using the **$menu.mobileMenu** key in the [config/frontend/05.menu.config.ts]()
```typescript
import {IMenuConfig} from 'wlc-engine/modules/menu';
export const $menu: IMenuConfig = {
    mobileMenu: {
        items: [
            'mobile-menu:sportsbook',
            'mobile-menu:promotions',
        ],
    },
};
```

If you want to add a brand new button, which configuration is not stored in the engine, then, for example, specify the button settings as an object
```typescript
import {IMenuConfig} from 'wlc-engine/modules/menu';
export const $menu: IMenuConfig = {
    mobileMenu: {
        items: [            
            'mobile-menu:sportsbook',
            'mobile-menu:promotions',,
            {
                name: gettext('Tournaments'),
                type: 'sref',
                class: 'tournaments',
                params: {
                    state: {
                        name: 'app.tournaments',
                        params: {
                            title: 'Tournaments',
                        },
                    },
                },
            },
        ],
    },
};
```

# 3.2.5.3. Мобильное меню

## Список доступных элементов меню
В файле [src/modules/menu/system/config/mobile-menu.items.config.ts]() в константе **wlcMobileMenuItemsGlobal** содержатся алиасы кнопок. У каждого алиаса свои настройки для кнопки меню.

```typescript
export const wlcMobileMenuItemsGlobal: MenuParams.IMenuItemsGlobal = {
    'mobile-menu:sportsbook': {
        name: gettext('Sportsbook'),
        type: 'sref',
        icon: 'sportsbook',
        class: 'sportsbook',
        params: {
            state: {
                name: 'app.sportsbook',
                params: {},
            },
        },
    },
    'mobile-menu:promotions': {
        name: gettext('Promotions'),
        type: 'sref',
        icon: 'promotions',
        class: 'promotions',
        params: {
            state: {
                name: 'app.promotions',
                params: {},
            },
        },
    },
```

## Дефолтная конфигурация меню
  Используя алиасы кнопок, можно выстраивать конфигурацию основного меню. В файле [src/modules/menu/system/config/menu.config.ts]() хранится дефолтная конфигурация мобильного меню.
```typescript
import {IMenuConfig} from '../interfaces/menu.interface';
export const menuConfig: IMenuConfig = {    
    mobileMenu: {
        items: [
            'mobile-menu:sportsbook',
            'mobile-menu:promotions',
            {
                parent: 'mobile-menu:info',
                items: [
                    'mobile-menu:privacy-policy',
                    'mobile-menu:responsible-game',
                    'mobile-menu:fair-play',
                    'mobile-menu:games-rules',
                    'mobile-menu:terms-and-conditions',
                ],
            },
        ],
        icons: {
            folder: 'wlc/icons/european/v1',
            use: false,
        },
    },
};
```

## Переназначение мобильного меню в проекте
**Важно:**
  Отображение категорий (к примеру Casino, Live Casino) в мобильном меню конфигурировать не нужно.
  Эти кнопки будут созданы автоматически на основе информации о категориях из **fundist** и будут вставлены в начало меню.

В проекте переназначается по ключу **$menu.mobileMenu** в файле [config/frontend/05.menu.config.ts]()
```typescript
import {IMenuConfig} from 'wlc-engine/modules/menu';
export const $menu: IMenuConfig = {
    mobileMenu: {
        items: [
            'mobile-menu:sportsbook',
            'mobile-menu:promotions',
        ],
    },
};
```

Если нужно добавить совершенно новую кнопку, конфигурация которой не хранится в движке, то настройки кнопки следует указать в виде объекта, к примеру
```typescript
import {IMenuConfig} from 'wlc-engine/modules/menu';
export const $menu: IMenuConfig = {
    mobileMenu: {
        items: [            
            'mobile-menu:sportsbook',
            'mobile-menu:promotions',,
            {
                name: gettext('Tournaments'),
                type: 'sref',
                class: 'tournaments',
                params: {
                    state: {
                        name: 'app.tournaments',
                        params: {
                            title: 'Tournaments',
                        },
                    },
                },
            },
        ],
    },
};
```
