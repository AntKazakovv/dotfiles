# 3.3.1. Основное меню

## Список доступных элементов меню
Настройки меню хранятся в файле [src/modules/menu/system/config/menu.config.ts]()  
В файле [src/modules/menu/system/config/main-menu.items.config.ts]() в константе `wlcMainMenuItemsGlobal` содержатся алиасы кнопок. У каждого алиаса свои настройки для кнопки меню.
```typescript
export const wlcMainMenuItemsGlobal: MenuParams.IMenuItemsGlobal = {    
    'main-menu:promotions': {
        name: gettext('Promotions'),
        type: 'sref',
        class: 'promo',
        params: {
            state: {
                name: 'app.promotions',
                params: {},
            },
        },
    },
    'main-menu:contacts': {
        name: gettext('Contacts'),
        type: 'sref',
        class: 'contacts',
        params: {
            state: {
                name: 'app.contacts',
                params: {
                    slug: 'feedback',
                },
            },
        },
    },
```

## Дефолтная конфигурация меню
Используя алиасы кнопок, можно выстраивать конфигурацию основного меню. В файле [src/modules/menu/system/config/menu.config.ts]() хранится дефолтная конфигурация основного меню
```typescript
import {IMenuConfig} from '../interfaces/menu.interface';
 
export const menuConfig: IMenuConfig = {
    mainMenu: {
        items: [
            'main-menu:promotions',
            'main-menu:contacts',
        ],
        icons: {
            folder: 'wlc/icons/european/v1',
            use: true,
        },
    },
};
```

## Переназначение основного меню в проекте

**Важно:**
Отображение разделов категорий (к примеру **Casino**, **Live Casino**) в основном меню конфигурировать не нужно.  
Эти кнопки будут созданы автоматически на основе информации о категориях из **fundist**.

В проекте переназначается по ключу `$menu.mainMenu` в файле [config/frontend/05.menu.config.ts.]()
```typescript
import {IMenuConfig} from 'wlc-engine/modules/menu';
 
export const $menu: IMenuConfig = {
    mainMenu: {
        items: [
            'main-menu:home',
            'main-menu:contacts',
        ],
    },
};
```
Если нужно добавить совершенно новую кнопку, конфигурация которой не хранится в движке, то настройки кнопки следует указать в виде объекта, к примеру
```typescript
import {IMenuConfig} from 'wlc-engine/modules/menu';
 
export const $menu: IMenuConfig = {
    mainMenu: {
        items: [        
            'main-menu:home',
            'main-menu:contacts',
            {
                name: gettext('Tournaments'),
                type: 'sref',
                class: 'tournaments',
                params: {
                    state: {
                        name: 'app.tournaments',
                        params: {
                            title: 'Tournaments',
                        },
                    },
                },
            },
        ],
    },
};
```
