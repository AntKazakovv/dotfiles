# 3.2.5.1. Main menu

## List of available menu items
The menu settings are stored in the [src/modules/menu/system/config/menu.config.ts]() file   
The [src/modules/menu/system/config/main-menu.items.config.ts]() file contains the `wlcMainMenuItemsGlobal` constant with button aliases. Each alias uses own settings for menu buttons.
```typescript
export const wlcMainMenuItemsGlobal: MenuParams.IMenuItemsGlobal = {    
    'main-menu:promotions': {
        name: gettext('Promotions'),
        type: 'sref',
        class: 'promo',
        params: {
            state: {
                name: 'app.promotions',
                params: {},
            },
        },
    },
    'main-menu:contacts': {
        name: gettext('Contacts'),
        type: 'sref',
        class: 'contacts',
        params: {
            state: {
                name: 'app.contacts',
                params: {
                    slug: 'feedback',
                },
            },
        },
    },
```

## The default menu configuration
You can use the button aliases to build the configuration of the main menu. The [src/modules/menu/system/config/menu.config.ts]() file stores the default configuration of the main menu
```typescript
import {IMenuConfig} from '../interfaces/menu.interface';
 
export const menuConfig: IMenuConfig = {
    mainMenu: {
        items: [
            'main-menu:promotions',
            'main-menu:contacts',
        ],
        icons: {
            folder: 'wlc/icons/european/v1',
            use: true,
        },
    },
};
```

## Reassigning the main menu in your project

**Important:**
You don't need to set up the display of the category sections (e.g. **Casino**, **Live Casino**) in the main menu.  
These buttons will be created automatically based on category information from **fundist**.

The menu can be reassigned in your project using the `$menu.mainMenu` key in the [config/frontend/05.menu.config.ts.]() file
```typescript
import {IMenuConfig} from 'wlc-engine/modules/menu';
 
export const $menu: IMenuConfig = {
    mainMenu: {
        items: [
            'main-menu:home',
            'main-menu:contacts',
        ],
    },
};
```
If you want to add a brand new button, which configuration is not stored in the engine, then, for example, specify the button settings as an object
```typescript
import {IMenuConfig} from 'wlc-engine/modules/menu';
 
export const $menu: IMenuConfig = {
    mainMenu: {
        items: [        
            'main-menu:home',
            'main-menu:contacts',
            {
                name: gettext('Tournaments'),
                type: 'sref',
                class: 'tournaments',
                params: {
                    state: {
                        name: 'app.tournaments',
                        params: {
                            title: 'Tournaments',
                        },
                    },
                },
            },
        ],
    },
};
```

# 3.2.5.1. Основное меню

## Список доступных элементов меню
Настройки меню хранятся в файле [src/modules/menu/system/config/menu.config.ts]()  
В файле [src/modules/menu/system/config/main-menu.items.config.ts]() в константе `wlcMainMenuItemsGlobal` содержатся алиасы кнопок. У каждого алиаса свои настройки для кнопки меню.
```typescript
export const wlcMainMenuItemsGlobal: MenuParams.IMenuItemsGlobal = {    
    'main-menu:promotions': {
        name: gettext('Promotions'),
        type: 'sref',
        class: 'promo',
        params: {
            state: {
                name: 'app.promotions',
                params: {},
            },
        },
    },
    'main-menu:contacts': {
        name: gettext('Contacts'),
        type: 'sref',
        class: 'contacts',
        params: {
            state: {
                name: 'app.contacts',
                params: {
                    slug: 'feedback',
                },
            },
        },
    },
```

## Дефолтная конфигурация меню
Используя алиасы кнопок, можно выстраивать конфигурацию основного меню. В файле [src/modules/menu/system/config/menu.config.ts]() хранится дефолтная конфигурация основного меню
```typescript
import {IMenuConfig} from '../interfaces/menu.interface';
 
export const menuConfig: IMenuConfig = {
    mainMenu: {
        items: [
            'main-menu:promotions',
            'main-menu:contacts',
        ],
        icons: {
            folder: 'wlc/icons/european/v1',
            use: true,
        },
    },
};
```

## Переназначение основного меню в проекте

**Важно:**
Отображение разделов категорий (к примеру **Casino**, **Live Casino**) в основном меню конфигурировать не нужно.  
Эти кнопки будут созданы автоматически на основе информации о категориях из **fundist**.

В проекте переназначается по ключу `$menu.mainMenu` в файле [config/frontend/05.menu.config.ts.]()
```typescript
import {IMenuConfig} from 'wlc-engine/modules/menu';
 
export const $menu: IMenuConfig = {
    mainMenu: {
        items: [
            'main-menu:home',
            'main-menu:contacts',
        ],
    },
};
```
Если нужно добавить совершенно новую кнопку, конфигурация которой не хранится в движке, то настройки кнопки следует указать в виде объекта, к примеру
```typescript
import {IMenuConfig} from 'wlc-engine/modules/menu';
 
export const $menu: IMenuConfig = {
    mainMenu: {
        items: [        
            'main-menu:home',
            'main-menu:contacts',
            {
                name: gettext('Tournaments'),
                type: 'sref',
                class: 'tournaments',
                params: {
                    state: {
                        name: 'app.tournaments',
                        params: {
                            title: 'Tournaments',
                        },
                    },
                },
            },
        ],
    },
};
```
