# 3.2.1. Basic configuration

It is the basic configuration of a WLC project. (see the object's [interface](../../../interfaces/IBaseConfig.html)).

## Example of the modular window configuration

* [interface](../../../interfaces/IModalList.html)
* The description of the modular window is available via the [path](https://vcsxa.egamings.com/wlc/wlc-engine/-/blob/master/src/modules/core/components/modal/modal.params.ts) indicated in the **MODALS_LIST** variable

```typescript
export const $base: IBaseConfig = {
    site: {
        restrictRegistration: false, // disable registation
    },
    modals: {
        login: {
            config: {
                id: 'login',
                modifier: 'login-sign-up',
                modalTitle: gettext('Sign in now!'),
                component: TabSwitcherComponent,
                size: 'as',
                showFooter: false,
                dismissAll: true,
                componentParams: {
                    themeMod: 'with-banner',
                    tabs: {
                        'signIn': {
                            name: gettext('Sign in'),
                            startTab: 'signIn',
                            component: 'user.wlc-sign-in-form',
                            modifier: 'login',
                            modalId: 'login',
                        },
                        'signUp': {
                            name: gettext('Sign up'),
                            component: 'core.wlc-steps',
                            theme: 'signInUp',
                            modifier: 'signup',
                            modalId: 'signup',
                        },
                    },
                },
            },
        },
    },
};
```

## Verification setup

### Options

- **use** - Using a component.
- **selectModeFrom** - The maximum number of document types to load, after which the template with the selector will be displayed.
- **maxDocsCount** - The maximum number of documents available for download.
- **maxSize** - The maximum size of the uploaded file. There is also a check on the backing side.
  
```typescript
export const profileConfig: IProfileConfig = {
    verification: <IVerification>{
        use: true,
        selectModeFrom: 4,
        maxDocsCount: 5,
        maxSize: 4,
    }
}
```

## Contact page setup

### Params

- **phone** - Phone number.
- **email** - Mailing address.
- **estResTime** - Estimated response time.
- **timeUnits** - Time units.
- **socials** - Social media icons.
  
```typescript
export type TimeUnits = 'hours' | 'minutes';

export interface IContactsConfig {
    phone?: string;
    email?: string;
    estResTime?: number;
    timeUnits?: TimeUnits;
    socials?: ISocialItem[];
}
```

## Настройка страны и валюты

По умолчанию для селектора валюты выбирается первое значение из списка, а селектор страны не заполнен.  
Значения по умолчанию можно переназначить в [04.modules.config.ts](3.2.3.-modules-config.html).  
Для этого нужно скопировать необходимую форму и задать значение **value** ([интерфейс](../../../interfaces/ISelectCParams.html)) в селекторах страны и валюты.

### Включение автовыбора страны и валюты

Для включение автовыбора страны и/или валюты по геопозиции, нужно в [04.modules.config.ts](3.2.3.-modules-config.html) установить параметр **autoSelect: true** в нужных селекторах.  
Подстановку валюты для страны мы должны прописать в этом конфиге самостоятельно.

```typescript
export const $base: IBaseConfig = {
    registration: {
        selectCurrencyByCountry: {
            rus: 'RUB',
            deu: 'EUR',
            est: 'BTC',
        }
    }
}
```

## Настройка сортировки валюты

Возможность расставить валюту при регистрации в определенном порядке.
Можно указать как полностью конфиг, так и частично вывести валюты выше, остальные будут в порядке как и были до сиортировки.

### Включение сортировки

```typescript
import {IBaseConfig} from 'wlc-engine/modules/core';

export const $base: IBaseConfig = {
    registration: {
        currencySort: [
            'CAD',
            'USD',
            'RUB',
        ],
    },
};
```

## Настройка верификации

### Параметры

- **use** - Использование компонента.
- **selectModeFrom** - Количество типов документов для загрузки, после которого будет отображаться шаблон с селектором.
- **maxDocsCount** - Максимальное количество документов доступных для загрузки.
- **maxSize** - Максимальный размер загружаемого файла. Также есть проверка на стороне бэка.
  
```typescript
export const profileConfig: IProfileConfig = {
    verification: <IVerification>{
        use: true,
        selectModeFrom: 4,
        maxDocsCount: 5,
        maxSize: 4,
    }
}
```

## Settings of the mocks

### Params

- **base** - Contains default routes with default mocks.
- **custom** - Use custom routes with custom sources.

Usage: 

Base field takes mocks from engine - `src/system/mocks/`

```typescript
export const $base: IBaseConfig = {
    mocks: {
        base: {
            tournaments: true,
            wins: true,
        },
        custom: [
            {
                route: '/api/v1/profiles',
                source: '/static/mock-profile.json',
            },
        ],
    },
};
```

```typescript
export interface IMocksConfig {
    base?: {
        jackpots?: boolean;
        wins?: boolean;
        topWins?: boolean;
        bonuses?: boolean;
        tournaments?: boolean;
    },
    custom?: [ICustomMock, ...ICustomMock[]],
}

interface ICustomMock {
    route: string;
    source: string;
}

```
