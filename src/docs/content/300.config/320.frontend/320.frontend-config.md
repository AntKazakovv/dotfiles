# 3.2. Frontend configuration

## Состав

Обязательный состав конфигурации, присутутвует во всех проектах

* [02.layouts.config.ts](3.2.-frontend-config/3.2.2.-layouts-config.html) - _конфигурация слоёв (структуры страниц) WLC

* 04.modules.config.ts - _конфигурация модулей WLC проекта._
* index.ts - _индексный файл. Экспортирует все объекты конфигурации._
