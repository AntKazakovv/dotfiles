# 3.1. Backend config

Все настройки **Wlc-core** находятся в проекте по пути: **config/backend.**

## Состав конфигурации

* 0.site.config.php
* 1.language.config.php
* 2.fundist.config.php
* 3.db.config.php
* 4.mail.config.php

## 0.site.config.php

* Основные настройки сайта: адрес, имя, домен и т.д.
* Настройки окружения: dev, qa, test, prod.
* Настройка редиректа на мобильную версию.
* Настройки социальных сетей (используется для регистрации через соцсети).
* Другие настройки Полные данные по конфигурации смотри в документации на Wlc core.

## 1.language.config.php

* Настройка языков сайта
* Настройка системы определения языка посетителя сайта

## 2.fundist.config.php

* Настройки доступа к API fundist для разных окружений.

## 3.db.config.php

* Настройки доступа к основной базе данных сайта для разных окружений.

## 4.mail.config.php

* Настройки Email для отправки сообщений пользователям, техподдержке и т.д.
