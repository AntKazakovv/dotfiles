# 3.1. Backend config

All **Wlc-core** settings are available within the project in **config/backend.**

## Configuration contents

* 0.site.config.php
* 1.language.config.php
* 2.fundist.config.php
* 3.db.config.php
* 4.mail.config.php

## 0.site.config.php

* Basic website settings: address, name, domain, etc.
* Environment settings: dev, qa, test, prod.
* Settings for redirection to the mobile version.
* Social media settings (for registering via social media).
* Other settings For complete configuration details, please refer to the Wlc core documentation.

## 1.language.config.php

* Website language settings
* Settings of the user language detection system on the website

## 2.fundist.config.php

* Environment-specific access settings for API fundist

## 3.db.config.php

* Environment-specific access settings for the core database of the website

## 4.mail.config.php

* Email settings for sending messages to users, support, etc.

# 3.1. Backend config

Все настройки **Wlc-core** находятся в проекте по пути: **config/backend.**

## Состав конфигурации

* 0.site.config.php
* 1.language.config.php
* 2.fundist.config.php
* 3.db.config.php
* 4.mail.config.php

## 0.site.config.php

* Основные настройки сайта: адрес, имя, домен и т.д.
* Настройки окружения: dev, qa, test, prod.
* Настройка редиректа на мобильную версию.
* Настройки социальных сетей (используется для регистрации через соцсети).
* Другие настройки Полные данные по конфигурации смотри в документации на Wlc core.

## 1.language.config.php

* Настройка языков сайта
* Настройка системы определения языка посетителя сайта

## 2.fundist.config.php

* Настройки доступа к API fundist для разных окружений.

## 3.db.config.php

* Настройки доступа к основной базе данных сайта для разных окружений.

## 4.mail.config.php

* Настройки Email для отправки сообщений пользователям, техподдержке и т.д.
